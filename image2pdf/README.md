# 图片转 PDF

将一张或多张图片合并为一个 PDF 文件。

## 快速开始

1. **首次使用**：回到上级目录，双击 **`install.bat`** 安装依赖。
2. **转换图片**（任选一种方式）：

| 方式 | 说明 |
|------|------|
| **`拖入图片.vbs`** | **最推荐**。把多张图片拖到 `.vbs` 文件图标上（不要拖到文件夹空白处）。 |
| **`双击运行.bat`** | 双击后弹出文件选择框，可 Ctrl 多选图片。 |
| 拖到 **`双击运行.bat` 图标** | 支持多图；部分电脑对 `.bat` 拖放不稳定，优先用 `.vbs`。 |

上级目录还有 **`拖拽-图片转PDF.vbs`**，用法相同。

## 输出

- 单张图：生成与源文件同名的 `.pdf`（同目录）。
- 多张图：合并为一个 PDF，文件名以第一张图为准。

## 支持格式

jpg、jpeg、png、bmp、gif、tif、tiff、webp、heic 等常见图片格式。

## 命令行

```bat
cd ..
python core\pdf_tool.py image2pdf 图1.png 图2.png
python core\pdf_tool.py image2pdf --pick
```

转换过程会显示进度条（按张）。
